# CSS Specificity

CSS Specificity，通俗来说，指的是 CSS 规则的优先级。

## 优先级

由上至下，优先级逐级下降：

1. 行内样式：优先级最高，例如 `<h1 style="color:#fff">Heading 1</h1>`

2. id 选择器：优先级第二，例如 `#navbar`

3. 类选择器，属性选择器，伪类选择器：优先级第三，例如 `.large`、`:hover`

4. 元素选择器，伪元素选择器：优先级最低，例如 `::after`

## 计算优先级

- 行内样式 - 1000 点

- id 选择器 - 100 点

- 类选择器等 - 10 点

- 元素选择器等 - 1 点

如果优先级点数相等，则取后面的 CSS 规则。
